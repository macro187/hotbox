Workstation Setup
=================

Shell
-----

See `macro-setup-shell`


inputrc
-------

    #
    # ~/.inputrc
    #

    $include path/to/macro-workstation/conf/inputrc

    # ...


gitconfig
---------

    #
    # ~/.gitconfig
    #

    [include]
        path = path/to/macro-workstation/conf/gitconfig

    # ...


vimrc
-----

    "
    " ~/.vimrc or ~/_vimrc
    "

    source path/to/macro-workstation/conf/vimrc

    " ...
